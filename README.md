# 📊 Jenson USA SQL Data Analysis Project

## 🔍 Overview

This project focuses on analyzing retail data for Jenson USA using SQL. The objective is to generate actionable insights related to sales performance, customer behavior, product demand, and staff efficiency to support data-driven business decisions.

## 📌 Key Analysis

* Store-wise sales performance
* Cumulative sales trends over time
* Category-wise revenue analysis
* High-value customer identification
* Customer-store relationship analysis
* Staff performance evaluation
* Top-selling and unsold product analysis

## 💡 Key Insights

* Identified high-performing and underperforming stores
* Highlighted top-selling and unsold products for inventory optimization
* Recognized high-value customers for targeted strategies
* Detected performance gaps among staff members

## 🛠 SQL Techniques Used

* JOINs
* CTEs (Common Table Expressions)
* Window Functions
* Subqueries
* EXISTS

## 📁 Files

* SQL Project Presentation (.pdf)
* SQL Queries

## 🎯 Conclusion

This project demonstrates the application of SQL in solving real-world business problems by transforming raw data into meaningful insights that support effective decision-making.
